# 利用 chatGPT 编程——学习 Python

 库中的所有代码都来自 chatGPT，我只是想通过 chatGPT 学习 Python 和其他语言，仅此而已。

## py-ytd 
 py-ytd 是我用 chatGPT 编写的第一个程序，这是一个用于下载 YouTube 视频的 Python 脚本。

### 已实现:
 - 通过 pytube 下载 YouTube 视频。
 - 对于把Python 当作一门工具语言的新手而言一个友好的下载工具。
 - 批量下载多个 YouTube 视频。
 - 下载有年龄限制的 YouTube 视频。
